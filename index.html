UPI Transaction Categorization Using Hybrid ML Architecture

UPI processes 21+ billion transactions per month in India, but transaction descriptions are messy and unstandardized — "BUNDL TECHNOLOGIES PVT LTD" is Swiggy. We built a semi-supervised categorization pipeline that works without labeled training data.

Architecture: E5 transformer embeddings encode merchant descriptions into 768-dimensional semantic vectors. HDBSCAN discovers category clusters without requiring a predefined count, handles varying cluster densities, and explicitly marks ambiguous transactions as noise instead of forcing wrong categories.

PythonE5 EmbeddingsHDBSCANHuggingFacescikit-learnUMAP
